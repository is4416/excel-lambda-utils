# ✂️ SmartJoin ー Excel LAMBDA関数で指定範囲からCSV文字列を作成

指定範囲からCSV文字列を作成します。  
複数行を指定した場合、スピルにより結果を出力します。

**引数**

| 引数   | 型     | 説明                              |
| ------ | ------ | --------------------------------- |
| Rng    | Range  | 変換もとのセルを配列として指定    |
| Result | String | 1行ごと、スピルとして文字列を返す |

**備考**

- Result は戻り値です。引数としては不要です。
- `"` は、`""` に変換されます
- すべてのセルは、ダブルクォートにより囲われます

**コード**

```vb
= LAMBDA(Rng, BYROW(
  Rng,
  LAMBDA(
    Line,
    TEXTJOIN(",", FALSE, """" & SUBSTITUTE(Line, """", """""") & """")
  )
))
```

**使用例**

SmartJoin という名前でブックに登録しているものとします。

```vb
= SmartJoin(A1:C5)
```
