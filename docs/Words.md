# ✂️ Words ー Excel LAMBDA関数でスペースで区切られた文字を分割して取得

スペースで区切られた文字を分割して取得します。  
スペースが含まれた文字は取得できません。

**引数**

| 引数   | 型             | 説明               |
| ------ | -------------- | ------------------ |
| Text   | String         | 分割前の文字列     |
| Result | HSTACK(String) | 分割後の文字の配列 |

**備考**

- Result は戻り値です。引数としては不要です。
- スピルには対応していません。

**コード**

```vb
= LAMBDA(Text, LET(
  List, TEXTSPLIT(SUBSTITUTE(Text, "　", " "), " "),
  FILTER(List, List <> "")
))
```

**使用例**

Words という名前でブックに登録しているものとします。  
スピルには対応していません。  
配列の配列を処理できない都合上、MAPでラップすることもできません。

```vb
= Words(A1)
```
