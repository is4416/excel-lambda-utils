# TriangleArea

## 説明

辺の長さ、角度から三角形の面積を計算します  
角度は、度数を指定してください

## TriangleAreaSSS 使用例
3辺 (A, B, C) がわかっている場合

= TriangleAreaSSS(A, B, C)

## TriangleAreaSAS 使用例
2辺 (A, B) とその間の角度 (R) がわかっている場合

= TriangleAreaASA(A, R, B)

## TriangleAreaASA 使用例
1辺 (A) とその両端の角度 (LR, RR) がわかっている場合

= TriangleAreaASA(A, LR, RR)
